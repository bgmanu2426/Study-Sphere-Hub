# **App Name**: Study Sphere Hub

## Core Features:

- User Authentication: User authentication through login and signup.
- Course Selection: Home page with selection of scheme (2022, 2021), branch (to be listed later, 5 initially), year (1, 2, 3, 4), and semester (1-8).
- Resource Display: Display notes and question papers based on the selected scheme, branch, year, and semester.
- Resource Download: Allow users to download notes and question papers for each subject. Include all subjects from VTU syllabus.
- AI Chatbot: AI powered chatbot that will tool use provided resources for responding to student queries about VTU, its courses, and syllabus.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2), evoking trust and competence. Blue is often associated with technology and knowledge.
- Background color: A light, desaturated blue (#E5F5F9) to ensure readability and reduce eye strain.
- Accent color: A warm yellow (#FFC107) to highlight important information and calls to action. Yellow creates contrast with the blue primary color and adds a sense of energy.
- Body and headline font: 'Inter' (sans-serif) for clear readability and a modern feel.
- Use clear and concise icons from a library like Font Awesome or Material Icons to represent different sections and actions.
- A clean, card-based layout for displaying resources. Prioritize information hierarchy for easy navigation.
- Subtle animations on page transitions and data loading to enhance the user experience without being distracting.
